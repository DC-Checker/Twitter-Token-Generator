                    TWITTER ACCOUNT GENERATOR
                         Quick Start Guide

Welcome! This tool generates Discord accounts automatically using various
captcha solvers and temporary email services.


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  GETTING STARTED
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Run Discord_Generator.exe
2. The tool will open - no installation needed


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  FIRST TIME SETUP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Before you can generate accounts, you need to add:

  ➤  API Keys for captcha solvers and email services
  ➤  Proxies (to avoid IP bans)

To set these up:
  1. Click "API & Proxies" tab at the top
  2. Scroll down and enter your API keys in the boxes
  3. Add proxies in the format: http://user:pass@ip:port
     (one per line in the big text box)
  4. Click "Save Configuration" button


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  WHERE TO GET API KEYS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  • Botwitter: https://castle.botwitter.com/

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  GENERATING ACCOUNTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to "Generator" tab

2. Configure your settings:
   ┌────────────────────────────────────────────────┐
   │  Threads: How many accounts to create at once │
   │           Start with 5-10 for testing          │
   │           Max: 300 threads                     │
   │                                                │
   │  Castle: Which captcha solver to use           │
   │  Email: Which email service to use             │
   │  FunCap: Which arkose solver to use            │
   │                                                │
   │  Swap Email: Change email after registration   │
   │              (only works with CyberTemp)       │
   │                                                │
   │  Reuse Email: Use same emails again            │
   │               (only works with Anymessage)     │
   └────────────────────────────────────────────────┘

3. Click "▶ Start Generation"

4. Watch the logs as accounts are created

5. Accounts get saved automatically to "registered.txt"


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  VIEWING YOUR ACCOUNTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Click the "Accounts" tab to see all generated accounts in a nice table.

You can:
  • Search for specific accounts
  • Copy details to clipboard
  • Export accounts
  • View stats (success/failed counts)

All accounts are saved in "registered.txt" in this format:
  username:password:email:emailid:ct0:authtoken


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  STOPPING GENERATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Click "⏹ Stop Generation" button

You'll see two options:
  • Graceful Stop: Let current accounts finish, then stop
  • Force Stop: Stop everything immediately

Any completed accounts will always be saved, even with force stop.

  1. Click the "⚙" settings icon (top right)
  2. Enable "Debug Mode"
  3. Check the logs for detailed error messages

  TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: No accounts generating
  ➜ Check you added API keys in "API & Proxies" tab
  ➜ Make sure proxies are working
  ➜ Enable debug mode to see detailed errors

Problem: High failure rate
  ➜ Try different captcha solver
  ➜ Check if your proxies are banned
  ➜ Lower thread count (5-10 threads)
  ➜ Switch to a different email service

Problem: API key errors
  ➜ Make sure you copied the full API key
  ➜ Check if you have balance on the service
  ➜ Some services need activation before first use

Problem: Proxy errors
  ➜ Test your proxies on another site first
  ➜ Format must be: http://user:pass@ip:port
  ➜ Make sure no extra spaces or blank lines

Problem: Force stop not working
  ➜ It now stops within 1 second
  ➜ All completed accounts are still saved

  TIPS FOR BEST RESULTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ Use good quality residential proxies (not free ones)
✓ Start with low threads (5-10) and increase if stable
✓ Pick one reliable captcha solver and stick with it
✓ CyberTemp is free but may be slower than paid email services
✓ Keep the tool updated for best compatibility
✓ Save your config regularly
✓ Check logs for any errors

  IMPORTANT NOTES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
